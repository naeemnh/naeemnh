# CLI Use Cases

## Primary Use Cases

### 1. Navigation & Exploration
- **Scenario**: User wants to browse portfolio sections in a terminal-like interface
- **Goal**: Navigate between different sections (work, about, blog, contact)
- **Commands**: `cd`, `ls`, `cat`, `help`

### 2. Viewing Content
- **Scenario**: User wants to read about work experience, projects, or blog posts
- **Goal**: Display formatted content in terminal style
- **Commands**: `cat`, `view`, `read`, `show`

### 3. Interactive Discovery
- **Scenario**: User wants to explore available commands and features
- **Goal**: Discover functionality through help system and autocomplete
- **Commands**: `help`, `man`, `tab` completion

### 4. Quick Information Access
- **Scenario**: User wants quick access to contact info, skills, or links
- **Goal**: Fast retrieval of specific information
- **Commands**: `whoami`, `contact`, `skills`, `links`

### 5. Fun/Engagement
- **Scenario**: User wants an engaging, memorable experience
- **Goal**: Create a unique, interactive portfolio experience
- **Features**: Easter eggs, animations, ASCII art, interactive commands

## Secondary Use Cases

### 6. Project Details
- **Scenario**: User wants detailed information about a specific project
- **Goal**: View project details, tech stack, links
- **Commands**: `project <name>`, `ls projects/`, `open <project>`

### 7. Blog Reading
- **Scenario**: User wants to read blog posts in terminal format
- **Goal**: Display blog content with proper formatting
- **Commands**: `blog`, `read <slug>`, `ls blog/`

### 8. Contact Interaction
- **Scenario**: User wants to send a message or get contact information
- **Goal**: Access contact form or information
- **Commands**: `contact`, `email`, `send`

### 9. Resume/CV Download
- **Scenario**: User wants to download resume
- **Goal**: Trigger download or view resume
- **Commands**: `resume`, `cv`, `download resume`

### 10. Social Links
- **Scenario**: User wants to access social media profiles
- **Goal**: Open or display social links
- **Commands**: `github`, `linkedin`, `social`, `links`
